# Content Strategy Engine

Sistema inteligente para sugerir posts, analisar público e recomendar melhores horários para Instagram, TikTok e LinkedIn.

## Estrutura do Projeto
- `src/` — código principal
- `tests/` — testes
- `scrapers/` — captura de tendências
- `audience_analyzer/` — análise de público
- `posting_time_optimizer/` — melhores horários
- `suggestion_engine/` — geração de ideias
